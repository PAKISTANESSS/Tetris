## Features

- **WebGL Rendering** - Pixel-perfect graphics with smooth animations
- **Classic Gameplay** - Standard Tetris mechanics with 7-bag randomizer
- **Sound Effects** - Procedural audio using Web Audio API
- **Haptic Feedback** - Vibration support for mobile devices
- **Responsive Controls** - Keyboard support with customizable key bindings
- **Modern UI** - Dark neon arcade theme with clean HUD
